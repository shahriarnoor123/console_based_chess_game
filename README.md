Project Title:
Console-Based Chess Game in C

Description:
This project is a console-based chess game implemented in the C programming language. It
simulates a standard two-player chess game (White vs Black) played through the terminal.

Key Features:
• 8×8 chess board
• All standard chess pieces
• Turn-based gameplay
• Move validation
• Check and Checkmate detection
• Pawn promotion
• Text-based interface

How to Run the Project:
Prerequisites:
• C Compiler (GCC recommended)
• Terminal / Command Prompt

Compilation:
gcc chess.c -o chess

Run the Program:
./chess (Linux/macOS)
chess (Windows)

How to Play:
Enter moves in the format: e2 e4
Type 'exit' to quit the game.

End of Game:
The game ends when checkmate occurs or the player exits.
